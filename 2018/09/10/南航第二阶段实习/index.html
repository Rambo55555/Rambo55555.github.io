日志

09-10

重回南航，被分到商务研发部。还是做JavaWeb的开发，在国庆前要完成一个前面师兄未完成的系统。导师在上午详细介绍了信息中心的构成和历史由来，然后说他几乎都在在各个部门待过，告诉我们不要只专注于程序编码，对一些其它方面也要注重。下午看了系统的文档，是做一个管理外包人员的系统，感觉系统还是有点复杂，没用过SpringBoot，微服务也不了解。要学的还有很多，当下不能浮躁，一步一步来，争取在这段时间完成好工作，学到有用的并掌握下来。

09-11

今天对SpringBoot做了入门学习，下载了STS，但是一个简单的Demo都跑不起来，原因好像又是maven仓库下包下不下来，不知道是什么原因导致的，很烦。下午组里被分了几台旧电脑，很不幸，我被分到的内存只有2G，还开不了机，GG，明天看能不能换一台。

09-12

​ 今日上午主要是给公司的旧电脑重装了Win10，奔腾的CPU，装完之后调了调性能优化了一下，运行速度还能接受。下午旁听了商务研发处的编码规范会，了解了一些常见的编码问题。代码规范：代码风格、代码质量、代码安全。复习了面向对象的七大原则，单一职责、开闭、里氏替换、合成复用、依赖倒转、接口隔离、迪米特。然后搭建了环境，学习了SpringBoot和Dubbo的使用，把要做的系统大致看了一下，SpringBoot还是很好懂的。晚上做了一个Demo来使用Zookeepr+Dubbo+SpringBoot，传到github上，学会了一些基本git命令。

git init

git add .

git status

git commit -m “Messages”

git remote add origin XXX.git

git push -u origin master

​ 昨晚着凉加上今天一天对着电脑中午没午睡，感冒了，鼻子很不舒服，脑袋有点(￣︶￣)↗ 涨。
